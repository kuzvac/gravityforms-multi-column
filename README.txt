Adds the ability to create multi-column forms.

== Description ==

This plugin adds a button to Form editor sidebar. Which creates a new column for the form. You can chang Field Label, or add custom css class, which will be added to a new column.

= Further Reading =

* [Official GitHub Repository](https://github.com/kuzvac/gravityforms-multi-column)

== Installation ==

1. Upload "gravityforms-multi-column" to "plugins" directory
2. Activate the plugin through the "Plugins" menu in WordPress
5. Add "New Column" field into forms which you desire to create new column of form in Gravity Forms Form Editor
